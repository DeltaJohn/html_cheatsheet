HTML Cheatsheet

Tables

Table Tags

Tag Name Description
<table> Table The wrapper for the entire table
<thead> Table Header Sets the row of cells to contain column Headers
<tbody> Table Body Actual table data in each of the cells
<tr> Table Row Sets a row of cells
<th> Header cell Defines info in that row of cells as containing column header names
<td> Table Data Defines info in that row of cells as containing data
<tfoot> Footer Contains the table footer info

Table Attributes

Attribute Name Description
colspan="X" Colspan Sets how many columns a td should span
rowspan="X" Rowspan Sets how many rows high a td should be
